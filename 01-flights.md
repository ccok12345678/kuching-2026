# 航班資訊

## 航線概覽

古晉通常需自台灣經 **吉隆坡（KUL）** 轉機前往，較常見的組合為：

- **馬來西亞航空（Malaysia Airlines）**：票務與行李轉掛相對完整
- **亞洲航空（AirAsia）**：價格通常較低，但需留意是否分段出票與行李加購

## 票價參考

| 項目 | 價格 |
|------|------|
| 來回機票（每人） | NT$10,000～18,000 |
| 來回機票（兩人） | NT$20,000～36,000 |
| 托運行李加購（廉航常見） | 約 MYR 60～120（約 NT$420～840）／段 |
| 機上餐食加購（廉航常見） | 約 MYR 15～30（約 NT$105～210）／人 |

## 購票建議

1. 建議至少 **出發前 2～3 個月** 開始比價。
2. 若選擇廉價航空，需把 **托運、選位、餐食、保險** 一併納入總價比較。
3. 年底接近馬來西亞學校假期，票價可能上漲，越早訂越有機會拿到合理價格。
4. 如分開購買兩段機票，轉機緩衝時間建議拉長，避免前段延誤影響後段。

## 機場與轉機提醒

- 桃園機場建議提早 **3 小時** 到場辦理。
- 若在吉隆坡需要重新提領行李再報到，務必確認航廈是否一致。
- 抵達古晉後仍需通過 **砂拉越入境審查**，即使已在吉隆坡入境馬來西亞也一樣。
- 雨季偶有班機延誤，建議第一天不要排太滿。
