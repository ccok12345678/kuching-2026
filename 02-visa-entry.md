# 簽證與入境

## 台灣旅客基本規則

| 項目 | 說明 |
|------|------|
| 簽證 | 持中華民國護照可免簽入境馬來西亞，停留通常不超過 90 天 |
| 護照效期 | 入境時建議至少剩餘 6 個月以上 |
| 回程證明 | 建議備妥回程機票與住宿訂單 |
| 入境資料 | 建議事先填寫 MDAC（Malaysia Digital Arrival Card） |

## 砂拉越入境重點

砂拉越（Sarawak）有獨立移民管理制度，因此即使在吉隆坡完成馬來西亞入境，抵達古晉仍可能需要再次接受砂拉越入境查驗。

### 建議隨身準備

- 護照正本
- 回程或續程電子機票
- 第一晚住宿資訊
- 簡要行程表
- 緊急聯絡人資訊

## 抵達流程簡表

1. 抵達古晉機場後依指示前往移民查驗區。
2. 出示護照與必要的住宿、回程資訊。
3. 通過後再提領行李或前往交通接駁區。
4. 若有表單或電子資料要求，依現場最新規定填寫。

## 實務提醒

- 建議把住宿地址與飯店電話存成手機截圖，避免現場網路不穩。
- 若使用電子登機證，也建議保留紙本或 PDF 備份。
- 入境問答通常簡單，重點是 **住宿地點、停留天數、返程日期**。
- 政策可能調整，出發前 1～2 週應再次確認官方公告。
