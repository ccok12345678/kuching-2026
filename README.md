# 2026 古晉（Kuching）自由行主目錄

本文件作為本次 **台灣 TPE → 馬來西亞古晉（Kuching, Sarawak）** 旅程的主目錄，方便快速掌握旅行重點並查閱各主題的詳細資訊。

## 旅行摘要

- **旅行日期：** 2026/11/28（六）～2026/12/05（六）
- **天數：** 8 天 7 夜
- **出發地：** 台灣桃園國際機場（TPE）
- **目的地：** 馬來西亞砂拉越州古晉（KCH）
- **建議旅行風格：** 城市漫步、文化體驗、自然生態、美食探索
- **匯率抓法：** 1 MYR ≈ 7 TWD（台幣）
- **整體預算參考：** 約 NT$52,000～94,000／2 人

## 行程摘要

| 天數 | 日期 | 行程摘要 |
|------|------|----------|
| Day 1 | 11/28（六） | 台灣出發，經吉隆坡轉機抵達古晉，入住飯店後前往水岸或夜市用餐 |
| Day 2 | 11/29（日） | 古晉市區步行探索：博物館、水岸、老街、馬來村落 |
| Day 3 | 11/30（一） | 前往砂拉越文化村與達馬里海灘，安排文化表演與海邊散步 |
| Day 4 | 12/01（二） | 峇哥國家公園一日遊，觀察長鼻猴、海岸地形與雨林生態 |
| Day 5 | 12/02（三） | Semenggoh 紅毛猩猩保育中心、老街購物、砂勞越河遊船 |
| Day 6 | 12/03（四） | 古晉美食主題日，安排叻沙、乾撈麵、海鮮與咖啡店巡禮 |
| Day 7 | 12/04（五） | 自由活動與採購伴手禮，可補足前幾天未完成景點 |
| Day 8 | 12/05（六） | 退房後前往機場，經吉隆坡轉機返回台灣 |

## 出發前建議先完成

- 確認護照效期至少還有 6 個月
- 預先比價來回機票與住宿
- 下載 Grab、Google Maps、Booking.com / Agoda
- 準備少量馬幣現金與可海外刷卡的信用卡
- 先查好 MDAC 與砂拉越入境流程
- 雨季期間預留彈性行程，避免戶外活動受天氣影響
