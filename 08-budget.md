# 預算估算

> 以下以 **2 人、7 晚、8 天、中等消費** 為估算基準。

## 主要費用

| 項目 | 估算金額 | 補充說明 |
|------|----------|----------|
| 來回機票 | NT$20,000～36,000 | 依航空公司、是否加購行李而定 |
| 住宿 | NT$15,000～25,000 | 以每晚 MYR 200～350（約 NT$1,400～2,450）估算 |
| 餐食 | NT$8,000～15,000 | 約每人每日 MYR 50～80（約 NT$350～560） |
| 景點門票 | NT$3,000～5,000 | 含文化村、峇哥、Semenggoh 等 |
| 市區與近郊交通 | NT$3,000～5,000 | Grab、渡輪、景點接駁 |
| 購物與伴手禮 | NT$3,000～8,000 | 視個人需求 |
| **合計** | **NT$52,000～94,000** | 約每人 NT$26,000～47,000 |

## 每日現金花費概念

| 類別 | 每日抓法 |
|------|----------|
| 早餐 + 午餐 + 晚餐 | 每人約 MYR 50～80（約 NT$350～560） |
| 市區短程交通 | 每日約 MYR 10～30（約 NT$70～210） |
| 小吃 / 飲料 / 點心 | 每人約 MYR 10～20（約 NT$70～140） |

## 建議帶多少現金

- 若大多使用 Grab 與信用卡，可先準備 **MYR 300～500（約 NT$2,100～3,500）** 現金／2 人。
- 若打算多逛市場、小吃攤、買小物，建議提高到 **MYR 600～800（約 NT$4,200～5,600）**。

## 控制預算的方法

1. 住宿集中在同一區域，減少換飯店成本。
2. 早餐與部分午餐以在地咖啡店解決，價格較友善。
3. 海鮮與遊船安排 1～2 次即可，不必天天高消費。
4. 優先購買可攜帶、單價不高的特色伴手禮，如黑胡椒、咖啡、醬料。
